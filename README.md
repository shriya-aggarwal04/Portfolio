#  Shriya Aggarwal – Portfolio Website

A personal portfolio website built using **React** and **Vite** to showcase projects, skills, and contact information.  
This portfolio highlights real-world projects like the Uber Ride Booking App and Wanderlust (Airbnb clone) and represents professional work in web development.

---

##  Project Overview

This portfolio website is designed to demonstrate frontend development skills using modern tools and frameworks.  
It features multiple sections such as About, Skills, Projects, and Contact — all built with React and styled with Tailwind CSS.

---

##  Tech Stack

- **React** – UI components  
- **Vite** – Dev server and build tool  
- **Tailwind CSS** – Utility-first styling  
- **JavaScript, HTML, CSS**

---

##  Key Features

- Modern, responsive UI across devices  
- Clean, component-based React structure  
- Sections include About, Skills, Projects, and Contact  
- Easy to extend for new content or sections  
- Perfect for showcasing development work

---

##  How to Use This Project

To use or customize this portfolio:

1. **Fork the repository** to your GitHub account  
2. **Clone** your fork to your local machine  
3. Install dependencies and set up the project  
4. Replace placeholders with your own content (projects, links, images)  
5. Customize styles or add new sections  
6. Deploy the site to a hosting service like **Vercel**, **Netlify**, or **GitHub Pages**

**Note:** You’ll need basic React knowledge to edit components such as `src/App.jsx` or other files under `src/`
